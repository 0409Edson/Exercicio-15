# Life Matriz

## Deploy Automático

Este projeto está configurado para deploy automático no Vercel.

### Como publicar:

```bash
# Primeira vez
npx vercel

# Atualizações (automático via GitHub)
git add .
git commit -m "Atualização"
git push
```
